Create System Administrator

Create a business-level system administrator. They will have access to manage tenants and users but will NOT have access to Django Admin panel.

Note: System Admins should be created via Django Admin. Go to: Django Admin → Users → Add User
  1. Set Role to: System Administrator
  2. Leave Tenant field blank
  3. Do NOT check "Superuser status"
  4. Do NOT check "Staff status" (unless you want admin access)
